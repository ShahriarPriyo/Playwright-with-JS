## Steps and Necessary Commands

1. Install NPM
2. Install Playwright
 ```bash
npx playwright install
```
3. Run Test
```bash
npx playwright test
```
4. Run tests in headed mode:
```bash
npx playwright test --headed
```
5. Run test in UI mode:
```bash
npx playwright test --ui
```
6. Show report:
```bash
npx playwright test show-report
```
